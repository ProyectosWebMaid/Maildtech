# MaildTech - Landing Page Pro

Proyecto listo para subir a GitHub Pages.

## Archivos
- `index.html`
- `styles.css`
- `script.js`

## Cómo probarlo en tu PC
1. Descomprime el ZIP.
2. Abre `index.html` con doble clic.

## Cómo subirlo a GitHub Pages
1. Crea un repositorio en GitHub, por ejemplo: `Maildtech`
2. Sube estos 3 archivos al repositorio.
3. Ve a **Settings > Pages**
4. En **Build and deployment**, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main` y carpeta `/root`
5. Guarda y espera 1 o 2 minutos.

Tu página quedará publicada en una URL tipo:
`https://tuusuario.github.io/Maildtech/`
