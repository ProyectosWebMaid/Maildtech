## Cómo agregar nuevos proyectos
1. Abre `proyectos.html`
2. Duplica un bloque `<article class="solution-card reveal project-showcase"> ... </article>`
3. Cambia:
   - título
   - descripción
   - etiquetas
   - enlaces de los botones
