## Qué subir a GitHub
Sube y reemplaza estos archivos en tu repo:
- index.html
- proyectos.html
- demo-corporativa.html
- demo-dashboard.html
- styles.css
- script.js

## Cómo agregar más proyectos
1. Abre `proyectos.html`
2. Duplica una tarjeta
3. Cambia texto y enlaces
4. Si quieres otra demo temporal, duplica una de las páginas demo
